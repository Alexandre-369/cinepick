# CinePick

CinePick is a lightweight movie decision app for people who spend too long choosing what to watch.

It combines mood-based recommendations, objective filters, a roulette mode, Letterboxd/Filmow CSV imports, and optional TMDb catalog loading for official posters and a larger movie base.

## Features

- Mood-based movie recommendations
- Roulette mode for indecisive nights
- Filters for genre, duration, decade, country, director/vibe, and minimum rating
- Local CSV import for watched movies and taste signals
- Optional TMDb integration using a user-provided read token
- Static Vercel deployment

## Local Use

Open `index.html` directly in a browser.

No build step is required.
